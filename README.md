# 🤖 TradingView Day Trader AI

AI-tool voor automatische day trading signalen op TradingView.

## Installation
1. `pip install -r requirements.txt`
2. `python main.py`

## Gebruik
- Open TradingView
- Start de AI met "🚀 START AI" knop
- Monitor de trading signalen in het log venster
- Signalen: BUY 📈 en SELL 📉 gebaseerd op RSI en MACD indicatoren

## Features
- ✅ Real-time screen capture van TradingView
- ✅ Technische indicatoren (RSI, MACD, Bollinger Bands)
- ✅ Automatische BUY/SELL signalen
- ✅ Live GUI dashboard
- ✅ Trade logging naar JSON
